# 소규모 스타트업 경영지원 자동화 — 7일 성공안

## 결론
소규모 스타트업에서 일주일 안에 성공시킬 수 있는 경영지원 자동화는 **법적 계산 자동화**가 아니라,
**반복 입력 제거 + 현금흐름 가시화 + 마감 누락 방지**에 집중해야 한다.

## 최종 채택 3개

### 1) 영수증 OCR → 지출 시트 자동입력 → 런웨이 자동계산
- 이유: 대표가 가장 자주 봐야 하는 건 현금 소진 속도와 잔고다.
- 효과: 경비 처리와 캐시 추적이 한 번에 해결된다.
- 완료 기준:
  - 사진 업로드 또는 파일 첨부 시
  - 날짜/금액/거래처/카테고리가 시트에 자동 입력되고
  - 잔액/월평균 burn/런웨이 개월 수가 자동 갱신된다.

### 2) 표준문서 자동생성
- 대상: 계약서, 견적서, 기안서, 확인서 등 자주 쓰는 3종
- 이유: 소규모 조직은 반복 문서 복붙 낭비가 크다.
- 완료 기준:
  - 시트에 변수 입력
  - HWP/PDF 문서 1클릭 생성

### 3) 마감·의무 일정 자동알림
- 대상: 세금, 4대보험, 계약만료, 정부지원사업 공고
- 이유: 놓치면 바로 손실이 난다.
- 완료 기준:
  - Discord #경영지원 채널에 D-7 / D-3 / D-1 자동 알림
  - 중요 일정 한눈에 확인 가능

## D1~D7 실행표
- D1: 지출 시트 마스터 구조 확정, 기존 데이터 정리
- D2: OCR 입력 파이프라인 연결, 자동 매핑 테스트
- D3: 런웨이 계산식/카테고리 규칙 완성
- D4: 표준문서 템플릿 3종 정비
- D5: 문서 자동생성 연결, 실제 샘플 출력 테스트
- D6: 마감/정부지원사업/계약 만료 알림 세팅
- D7: 실사용 테스트 + 대표 데모 + 가이드 1장 배포

---

## 성공 판정 기준
- 대표가 시트만 열어도 현재 현금 상태와 런웨이를 즉시 본다.
- 영수증 처리 시간이 건당 10초 수준으로 줄어든다.
- 자주 쓰는 문서 3종이 복붙 없이 생성된다.
- 이번 주 내 실제 알림이 채널에 울린다.

---

## 대표 즉시 승인 3개
1. 지출 마스터 시트/런웨이 기준표를 단일 원본으로 쓴다.
2. 표준문서 3종(계약서/견적서/기안서) 템플릿을 고정한다.
3. #경영지원 채널을 마감·정부지원사업·계약 알림의 단일 수신 채널로 쓴다.
